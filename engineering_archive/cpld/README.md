# CPLD Digital Audio Router

## Overview

This directory contains the complete CPLD design used in the **RTAL Digital Audio Patchbay**.

The design is based on a **Xilinx XC9572-15-PC84** CPLD and implements the digital routing logic for multiple S/PDIF audio sources. Rather than processing audio data itself, the CPLD performs deterministic hardware routing between digital inputs and outputs under control of the system microcontroller.

The implementation demonstrates how a relatively small CPLD can replace a large amount of discrete TTL logic while providing extremely low latency and deterministic switching.

---

# Design Goals

* Hardware-based digital audio routing
* Near-zero switching latency
* Simultaneous routing to multiple destinations
* Reliable operation without CPU timing dependencies
* Simple parallel control interface
* Compact implementation using a XC9572 CPLD

# Digital Outputs

The CPLD controls routing to several independent destinations.

* DA Converter
* Digital Output
* Recorder 1
* Recorder 2
* DSP Interface
* PC Interface
* AUX Output
* Front Connector
* Error Output

Multiple destinations may be connected simultaneously depending on the selected routing configuration.

---

# Control Interface

The CPLD is controlled through a parallel control bus generated by the system microcontroller.

```
A1 ... A4
B5 ... B8
C9 ... C12
D13 ... D15
E16 ... E19
```

The control bus selects which source is connected to which output. All routing decisions are implemented directly inside the CPLD hardware, allowing deterministic operation independent of firmware execution time.

# Historical Background

The CPLD was developed as part of the RTAL Digital Audio Patchbay project to provide flexible routing between numerous S/PDIF sources and destinations.

At the time of development, implementing this functionality using a CPLD offered significant advantages over discrete logic:

* Reduced component count
* Higher reliability
* Easier maintenance
* Flexible routing logic
* Very low propagation delay

The design represents a typical example of professional digital audio hardware developed during the late 1990s and early 2000s.
